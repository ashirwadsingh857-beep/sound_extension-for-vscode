# VS Code Sounds 🎧

Turn your coding sessions into an immersive experience with **VS Code Sounds**. This extension provides instant auditory feedback for 7 different development events, helping you stay "in the zone" without always looking at the bottom of your screen.

Built by **Ashirwad Rakesh Singh**, Electronics & Telecommunication Engineering student at **SIT Lonavala**.

---

## 🚀 Key Features (7 Custom Sounds)

This extension monitors your environment and triggers specific high-quality sounds for:

1.  **Inline Errors (Red Lines):** Instant feedback the moment a syntax error appears in your code.
2.  **Inline Warnings (Yellow Lines):** Distinct sound for warnings or linting issues.
3.  **Terminal Success:** A satisfying "Success" sound when your script or task finishes with Exit Code 0.
4.  **Terminal Error:** A "Crash" sound if your code fails or exits with an error code.
5.  **Large Paste Alert:** Notifies you when a large block of code (>50 characters) is pasted.
6.  **New File Ignition:** A startup sound triggered the moment you open a blank file to start fresh.
7.  **Suggestion Accepted:** A unique "Pop" sound when you press **Tab** to accept an IntelliSense or Inline Suggestion.

---

## 🛠️ Configuration

The extension is designed to be **Instant**. There is no delay between an error appearing and the sound playing, giving you a truly responsive feel.

### Sound Map:
* `error_new.mp3` -> Editor Errors
* `warning.mp3` -> Editor Warnings
* `success.mp3` -> Terminal Success
* `error.mp3` -> Terminal Errors
* `paste_large.mp3` -> Large Content Pastes
* `new_file.mp3` -> New File Creation
* `suggestion.mp3` -> Accepting Suggestions (Tab)
